# My Perfume Store

A responsive e-commerce perfume store website built using **HTML, CSS, and JavaScript**. This project is designed to simulate a real online perfume shopping experience with a modern UI, responsive layout, and interactive features.

## Features

* Responsive design for desktop and mobile
* Modern perfume store interface
* Product showcase section
* Shopping cart UI
* Clean and user-friendly layout
* Interactive front-end components using JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Goals

The goal of this project is to gain hands-on experience in **front-end web development** and **e-commerce website design** by building a real-world online perfume store. This repository is part of my learning journey and portfolio development.

## Future Improvements

* Add backend functionality
* User authentication system
* Payment gateway integration
* Product filtering and search
* Admin dashboard
* AI-powered product recommendations

## Installation

1. Clone the repository:

```bash
git clone https://github.com/nicedevil02/my_perfume_store.git
```

2. Open the project folder.

3. Run the project by opening `index.html` in your browser.
